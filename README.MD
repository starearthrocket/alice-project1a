# Level 5 Static Website (HTML/CSS)

## Overview
This project is a 3-page static website built using custom HTML5 and CSS3. It demonstrates consistent navigation, structured layout, responsive design, and accessible markup.

## User Value
Users can:
- Navigate between pages using a clear main menu
- Read about available services/pricing
- Use an accessible contact form (demo) to draft an enquiry

## Pages
- Home (`index.html`)
- Services (`services.html`)
- Contact (`contact.html`)

## Technologies Used
- HTML5
- CSS3 (custom)

## Deployment (GitHub Pages)
1. Push the repository to GitHub.
2. Go to **Settings → Pages**.
3. Under **Build and deployment**:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/(root)`
4. Save and open the Pages URL.

## Local Development
Open `index.html` in a browser, or run using a simple local server (e.g., VS Code Live Server).
